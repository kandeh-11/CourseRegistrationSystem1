# Course Registration System

This project is a C# console application that simulates a basic course registration system. 
It allows users to view available courses and is being developed using sprint-based planning.

## Sprint 1 (February 16 – March 1)

### What was completed
- Created core classes such as Student and Course
- Set up the project structure in Visual Studio
- Displayed a list of available courses in the console

### Outcome
A working program that displays available courses with basic class structure completed.

## Technology Used
- C#
- .NET Console Application

## How to Run
Open the project in Visual Studio and click the Run button.
